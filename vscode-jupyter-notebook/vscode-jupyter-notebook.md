## Vscode Jupyter Notebook with VIM

- block
	- Insert block below: `b`
	- Insert block above: `a`
	- Copy block: `c`
	- Paste block: `v`
	- Cut block: `x`
	- Delete block: `dd`
	- Collaspes block: ???
- Movement
	- `j`, `k`
	- Go to the bottom `G`
	- Go to the top: `gg`
	
- Convert to Code block: `y`
- Convert to Markdown block: `m`

- revert action: `z`
